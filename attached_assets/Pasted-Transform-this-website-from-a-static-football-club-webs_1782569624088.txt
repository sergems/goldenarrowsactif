Transform this website from a static football club website into an immersive, interactive supporter platform that feels like a modern football ecosystem.
The website must feel alive every single day, not only on match days.
The design should use smooth animations, micro-interactions, real-time updates, dynamic backgrounds, modern transitions and engaging user experiences without sacrificing speed.
Every page should have something that encourages visitors to interact rather than simply read information.
